# 🚲 BEACON 사용설명서

> 이 설명서는 앱 안에도 들어 있어요 (📖 사용법 버튼).
> 사막에서 인터넷이 안 돼도 앱만 열면 언제든 다시 볼 수 있어요.

## 이 앱이 뭐냐면

인터넷이 **안 터져도** 되는 생존 도우미예요. 세 가지를 해요:

1. **다음 물/음식/마트까지 몇 km 남았나**
2. **해 지기 전에 거기 도착할 수 있나**
3. **내 마지막 위치를 기억했다가, 가족에게 문자로 알려주기**

지도는 없어요. 지도는 지금 쓰는 앱이 더 잘해요. 이 앱은 "판단"만 해요.

## 딱 한 번, 인터넷 될 때 (제일 중요!)

1. 사파리에서 링크를 열어요.
2. 아래 가운데 **공유 버튼(네모에 화살표 ↑)** 을 눌러요.
3. **"홈 화면에 추가"** 를 눌러요.
4. 바탕화면에 아이콘이 생겼어요. **앞으로는 꼭 이 아이콘으로** 열어요.
5. 빨간 칸에 **가족 전화번호**를 미리 넣어두세요 (한 번만 넣으면 저장돼요).

⚠️ 이걸 안 하면 인터넷 없는 사막 한가운데서 앱이 **안 열려요.** 꼭 해주세요.

## 매일 아침, 출발 전에

1. 아이콘 눌러서 앱 열기
2. **[📍 GPS로 내 위치 잡기]** 누르기 (하늘 보이는 곳에서)
3. 나오는 숫자 읽기 — 끝!

## 숫자 읽는 법

| 화면 | 뜻 |
|------|-----|
| 💧 다음 물 **32km** | 32km 더 가면 물이 있다 |
| 🌅 일몰까지 **3h 10m** | 3시간 10분 뒤에 해가 진다 |
| ✅ | 해 지기 전에 도착 가능 |
| ❌ | 오늘은 무리 — 욕심내지 말기 |
| ⚠️ | 확실하지 않은 정보라는 뜻. **물은 항상 넉넉하게!** |
| "해 졌음" | 오늘 라이딩은 끝. 무리하지 않기 🙂 |

## 🆘 위험할 때 (순서 꼭 기억!)

1. **000에 전화**해요. 안 되면 **112**. (신호가 없어 보여도 일단 시도!)
2. 전화가 아예 안 되면, iPhone이 **"위성 SOS"** 화면을 띄워요.
   하늘이 보이는 곳에서 화면 안내를 따라 폰을 움직이면 **인터넷 없이도** 구조 요청이 가요.
3. 빨간 칸의 **[📨 가족에게 위치 문자]** 를 눌러두세요.
   신호가 1칸이라도 잡히는 순간 문자가 나가요.
4. 내 폰이 완전히 먹통이면? **남의 전화를 빌려요.** 지나가는 차를 세우거나
   로드하우스 전화로 000에 신고할 때, **[📋 위치 복사]** 를 눌러 나온
   **좌표 숫자를 소리내어 읽어주세요** (또는 화면을 보여주세요).
   구조대는 이 숫자만 있으면 정확한 지점으로 올 수 있어요.

이 앱은 GPS를 잡을 때마다 위치를 **자동으로 저장**해요 (비행기 블랙박스처럼).
그래서 "마지막으로 어디 있었는지"는 항상 남아 있어요.

**솔직한 말:** 신호가 0칸이면 문자는 보통 못 나가요. 단, **iPhone 14 이상**이면
문자 앱이 **"위성으로 보내기"**를 띄워줘요 — 하늘 아래에서 안내를 따라 하면 신호 없이도 나가요.
그것도 안 되면 위성 SOS(2번)가 답이에요.

## 내 위치는 누가 볼 수 있나요?

**딱 한 사람 — 문자를 받는 사람만.** 이 앱은 서버가 없어요.
위치는 **이 폰 안에만** 저장되고, 문자를 보낼 때도 폰에서 가족에게 **직접** 가요.
앱을 만들어준 사람을 포함해 **그 누구도** 위치를 볼 수 없어요.

## 가족한테 미리 부탁할 것

> "내 문자에 🆘나 지도 링크가 오면 바로 열어봐.
> 그리고 **이틀 넘게 아무 소식 없으면** 호주 경찰(000)에 마지막 위치를 알려줘."

출발 전에 이 한마디를 해두면 앱이 진짜 안전망이 돼요.

## GPS가 안 잡히면

도로 표지판이나 지도에서 본 **km 숫자를 직접 입력**하면 똑같이 계산돼요.

## 뭔가 이상할 때

| 증상 | 해결 |
|------|------|
| 앱이 하얗게 나와요 | 홈 화면 **아이콘**으로 열었는지 확인 |
| 숫자가 이상해요 | km 직접 입력으로 바꿔보기 |
| GPS가 계속 실패해요 | 하늘이 트인 곳에서 다시 시도, 안 되면 km 직접 입력 |

## 이것만 기억해요

> **물 넉넉히. 해 지면 멈추기. ❌면 욕심내지 않기.**
> **위험하면: 000 → 위성 SOS → 가족 문자.**
> 앱은 도와줄 뿐, 판단은 항상 몸 상태가 먼저예요.
