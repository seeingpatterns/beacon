# 🚲 BEACON

호주 대륙을 자전거로 횡단하는 라이더를 위한 **완전 오프라인 생존 도우미** (PWA).

널라버처럼 통신이 없는 구간에서 딱 세 가지만 한다:

1. **다음 물/보급/마트까지 몇 km** — GPS 또는 km 수동 입력으로 계산
2. **일몰까지 남은 시간** — 해 지기 전에 어디까지 갈 수 있는지 판정
3. **마지막 신호** — 위치를 자동 기록(블랙박스)하고, 가족에게 보낼 위치 문자 초안 생성

## 원칙

- **서버 없음.** 위치는 폰 밖으로 나가지 않는다. 문자도 폰에서 가족에게 직접 간다.
- **의존성 0.** 파일 몇 개가 전부. 전체 코드를 한 사람이 읽을 수 있는 크기.
- **틀린 안내 금지.** 경로에서 20km 이상 벗어나면 계산하지 않는다. 미확인 데이터는 ⚠️로 표시한다.
- **정직한 한계.** 신호 0칸이면 문자는 못 나간다고 말한다. 진짜 응급은 000과 iPhone 위성 SOS.

## 사용

한 번 온라인에서 열고 **"홈 화면에 추가"** → 이후 완전 오프라인 동작.
자세한 사용법은 앱 안의 📖 버튼 또는 [docs/사용설명서.md](docs/사용설명서.md).

## 개발

```bash
python3 -m http.server 8765   # 로컬 실행
node --test tests/*.test.js   # 테스트 (의존성 없음, node 내장 러너)
```

경로 데이터(`route-data.json`)는 멜버른→퍼스 Eyre Hwy 기준. POI `verified: true`는
자전거 여행자 실측 출처와 대조 검증된 항목.
